# Q&A
本部分内容汇总了科赛网用户在使用K-Lab过程中遇到的问题，定期更新。

Q1：K-Lab提供什么样的计算资源？        
A1：K-Lab为每位用户分配2核8G的独享计算资源，为了让计算资源有效分配，用户单次使用时长限制为**2小时**，超过2小时 kernel 将自动断开，用户可以通过刷新手动连接 kernel 分配到新的计算资源，继续开展分析工作。
剩余可用时间在右上角显示。
